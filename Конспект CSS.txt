CSS - (Cascading Style Sheets) Каскадие таблицы стилей, язык для описания внешнего вида документа;

<link rel="stylesheet" href="ссылка"> - подключание файла стилей к HTML файлу;
 *.css - расширение файла стилей;

color - цвет;
font-size - размер текста, задается в px;

пример стилей для тега <p></p> - селектор по названию тега:
	p{
		color: #d2d2d2;
		font-size: 24px;
	}
пример стилей за конкретным id:
	#grut{
		color: red;
		...
	}

пример применения стилей в файле HTML к ссылке:
...
	<style>
		a {
		    ...
		}
	</style>
....
	<a href=""> ...</a>
...

Селекторы
*{...} - универсальный селектор, применяется ко всем элементам страницы;
.Name_class {...} - применяется к нескольким элементам;
#Name_ID {...} - обычно применяется к уникальному (одному) элементу;
тег[атрибут="значение"] {...} - применение селектора к тегам за атрибутами;
	пример на ссылке: a[href="ссылкa"] {...}

Пример селектора предок:
	...
	<div class="NAME">
		....
	</div>
....
.NAME тег {...} - название тегов к которым будет применяться своство, применятеться ко всем нащадкам;
.NAME > тег {...} - применяеться только к первым нащадкам;
	пример: ...
		<div class="name">
			<p>...</p>
			<p>...</p>
			<div><p>...</p></div> - этот параграф будет по умолчанию черны;
			<p>...</p>
		</div>
....
name > p {
	color: red;
}

пример сестринского селектора за ИД и тегом <span>:
	<h1 id="nme">...</h1>
	<span>...</span> - на этом теге селектор будет применяться;
	<span>...</span> - на этот тег сесринский тг не распространяеться;
...
	#name + span {...} 


тег:псевдокласс {...} - во время некоторого события выполняеться дествие;
	пример: a:hover {text-decoration: none; } - при наведении на ссылку подчеркивание изчезает;
песвдокласы: hover - при наведении;
	     active - при активиции;

тег::псевдо-элемент {...} - тзменяет как-то какой-то yчасток страницы;
	пример: h1::first-letter {
			color: red;
			font-size: 36px;
		} - первая буква заголовка красного цвета и 36px;

Верстка оформления страниц сайта
Box model - это модель которая описывает все блочные элементы в HTML, и у них есть следущие свойства:
	margin - внешний отступ;
	border - рамка;
	padding - внутрений отступ;
	content - сам контент;

теги селекторов: margin-left - отступ текста, задается в px;
		 margin: auto - автоматическая подборка растояний слева и справа;
		 padding - аналогично margin;

	width - ширина;
	height - высота;
	text-align - положение теста;

Структура сайта растротрена при изучении HTML5

text-shadow: red 0 0 10px; - красная тень вокруг букв;
clear: both; - выравнивание подвала;
text-shadow: black 1px 1px 0, black -1px -1px 0,
	     black -1px 1px 0, black 1px, -1px, 0; - черная линия вокруг букв, хорошо видно на большом тексте;
background: url(...); - фон картинка;

Использование шрифтов
font-family: Arial, Helvetica, sans-serif; - для вывода шрифта в браузере, на ПК должна быть установлена библиотека этого шрифта, именно по этому используют сразу несколько вариантов шрифтов;

Цветовые модели и единицы измерения 
color: цвет;
	способы задания цветов: #.....
				rgb(..., ..., ...)
					rgba(..., ..., ..., .5) полупрозрачный цвет;
				color
единицы измерения: px, %, em (1em = 100%), vw и vh (1% размера окна, соответственно ширины и высоты, 1/100 размера, созданы для мобильных преложений);
